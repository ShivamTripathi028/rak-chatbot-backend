## Product Description

The RAK12039 is a Particle Matter Sensor module that is part of the RAKWireless WisBlock Sensor series. It is based on PMSA003I digital particle concentration sensor, which obtains the number of suspended particles in the air. It is interfaced via I2C bus and allows you to capture standard PM1.0, PM2.5, and PM10 values, as well as, a number of particles in micrometer on a 0.1Â L particle standard volume.
