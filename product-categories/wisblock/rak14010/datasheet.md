# RAK14010 WisBlock 3x4 Keypad Module Datasheet

## Overview

### Description

**RAK14010 WisBlock 3x4 Keypad Module** was designed to be part of a production-ready IoT solution and must be combined with a [RAK14004 WisBlock Keypad Module](https://docs.rakwireless.com/product-categories/wisblock/rak14004/overview/).

### Features

* **Module Specifications**
  * 3 x 4 matrix keypad
* **Size**
  * 43 mm x 44.99 mm
