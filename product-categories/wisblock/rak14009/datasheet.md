# RAK14009 WisBlock 3x3 Keypad Module Datasheet

## Overview

### Description

**RAK14009 WisBlock 3x3 Keypad Module** was designed to be part of a production-ready IoT solution and must be combined with a [RAK14004 WisBlock Keypad Module](https://docs.rakwireless.com/product-categories/wisblock/rak14004).

### Features

* **Module Specifications**
  * 3x3 matrix keypad
* **Size**
  * 33&nbsp;mm x 45&nbsp;mm
