## Product Description

RAK13006 is a CAN Bus communication module based on the MCP2518FD CAN controller and ATA6563 CAN transceiver both from Microchip. The MCP2518FD is the CAN chip that communicates to WisBlock Core MCU via SPI up to 17Â MHz SPI Clock Speed, and ATA6563 is the low-level physical layer chip that provides a physical connection with the CAN bus communication lines. It supports both CAN 2.0B and CAN FD with an arbitration bit rate up to 1Â Mbps. This WisBlock Interface module is ideal for industrial and automotive applications.
