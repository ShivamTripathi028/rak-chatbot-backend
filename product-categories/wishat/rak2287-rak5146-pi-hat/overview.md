## Product Description

**RAK2287/R–ê–ö5146 Pi HAT** is a converter board with a Raspberry Pi form factor that enables the RAK2287 SPI or RAK5146 SPI LPWAN concentrator modules to be mounted on the top of the Raspberry Pi.

This board integrates one (1) 40-pin female Pi HAT connector and one (1) mini PCIe connector. The RAK Pi HAT is compatible with the Raspberry Pi 3 Model B+ and Raspberry 4.

## Product Features

- Raspberry Pi form factor with a 40-pin compatible header
- Compatible with the Raspberry Pi 3 Model B+/Raspberry 4
- Mini PCIe connector on board
