# RAK7391 WisGate Connect Compatible Hardware

| Component                                | Compatibility                                                                                                                                                                                                                                                                                                  |
|------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Compute Module 4                         | All of them, lite and non-Lite versions.                                                                                                                                                                                                                                                                       |
| CM4 Heatsink                             | Optional but recommended. 
 Not compatible with Indoors Aluminum enclosure. 
 The enclosure doubles as heat-sink.                                                                                                                                                                                    |
| PoE Support                              | Optional (RAK9063A) 
 Included by default with the outdoors enclosure.                                                                                                                                                                                                                                    |
| OLED Screen                              | Optional (RAK1921) 
 Included by default with the indoors enclosures.                                                                                                                                                                                                                                     |
| Available connectors                     | All connectors are accessible except for the outdoor enclosure, where only the 1 Gb Ethernet port, USB Type-C, and serial console are accessible from outside.                                                                                                                                            |
| MiniPCIe Slot #1                         | RAK2247-SPI/USB LoRaWAN Concentrator 
 RAK2287-SPI/USB LoRaWAN Concentrator (optional GPS support) 
 RAK5146-SPI/USB LoRaWAN Concentrator (optional GPS support) 
 RAK8213 EG95-E/NA LTE Module 
 Miromico Mioty Edge Card 
 Z-Wave.Me mPCIe                                          |
| MiniPCIe Slot #2                         | RAK2247-SPI/USB LoRaWAN Concentrator 
 RAK2287-SPI/USB LoRaWAN Concentrator 
 RAK5146-SPI/USB LoRaWAN Concentrator 
 RAK8213 EG95-E/NA LTE Module 
 Miromico Mioty Edge Card 
 Z-Wave.Me mPCIe                                                                                        |
| MiniPCIe Slot #3                         | RAK2247-USB LoRaWAN Concentrator 
 RAK2287-USB LoRaWAN Concentrator 
 RAK5146-USB LoRaWAN Concentrator 
 RAK8213 EG95-E/NA LTE Module  
 Miromico Mioty Edge Card
 WiFi6 module (with M.2 to MiniPCIe adapter) 
 1/2 channels MiniPCIe to SATA converter 
 Z-Wave.Me mPCIe  |
| WisBlock Slot #1                         | All IO WisBlock Modules are compatible (not all may currently be supported in software)                                                                                                                                                                                                                                          |
| WisBlock Slot #2                         | All IO WisBlock Modules are compatible (not all may currently be supported in software)                                                                                                                                                                                                                                        |
| Raspberry Pi Hat Header                  | Alpha Network WiFi HaLow Hat                                                                                                                                                                                                                                                                                   |
| 3rd party Raspberry Pi Hat compatibility | Max hat height (not including the header & PCB thickness): 
 • 22 mm in the Indoors Aluminum enclosure 
 • 19 mm in the Indoors Sheet Metal enclosure 
 • 18 mm in the Outdoors enclosure with Antenna Board 
 • 44 mm in the Outdoors enclosure without Antenna Board |
| M.2 Slot                                 | 4G modules (EM060K-GL) 
 5G modules (RM520N-GL, RM505Q) 
 Up to 5-channels M.2 to SATA converter 
 M.2 B-keying NVMe drive (NOT SATA), see [SSD for WisGate Connect](https://store.rakwireless.com/products/ssd-for-rak7391)                                                                    |
| Antenna connectors                       | 2x iPEX to SMA pigtails included with Indoors enclosures. 
 Up to 6x SMA/RP-SMA connectors in Indoors enclosures. 
 2x iPEX to Type-N pigtails included with Outdoors enclosure.                                                                                                                     |
| Antenna board (GPS/LTE/WIFI/BT)          | Optional with Outdoors enclosure only.                                                                                                                                                                                                                                                                         |
