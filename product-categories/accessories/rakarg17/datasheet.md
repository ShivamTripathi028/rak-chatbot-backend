# 860-930MHz 5.8dBi LoRaWAN Antenna Black Datasheet

## Overview

> **Image:** Antenna Black Overview

### Features

- Frequency Range: **860~930 MHz**
- Max Gain: **5.9 dBi**
- VSWR: **≤ 2**
- High Efficiency
- Vertically Polarized Dipole
- N-type connector
- Easy to mount with the included Installation kit
