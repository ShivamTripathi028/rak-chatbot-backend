# WisBlock RAKBox-B6 Enclosure Installation Guide

## Assembly List

> **Image:** Assembly List

## Assembly

1. Fix the core and sensor modules to the WisBlock Base with an appropriate number of M1.2*3 screws. For assembly methods and precautions, refer to the [WisBlock Base Board Installation Guide](https://learn.rakwireless.com/hc/en-us/articles/26743966497431-How-To-Install-RAK5005-O-Baseboard/).

> **Image:** Attach Core and Sensor Modules

2. Put the battery in the space between four stand-offs of the enclosure.

:::tip NOTE
Ensure the battery dimensions are less than 65 mm x 44 mm x 8 mm to fit the enclosure.
:::

> **Image:** Battery Assembly

3. Fix the mounting plate onto the enclosure using tapping screws.

> **Image:** Fix Mounting Plate

4. Install the WisBlock modules with four M2.5*4 screws.

> **Image:** Fix WisBlock Base Board

5. Stick the antenna to the sidewall of the enclosure and connect the antenna wire to the IPEX connector.

> **Image:** Antenna Installation

6. Plug the terminal of the battery into the socket of the WisBlock module. Then, assemble the lid.

:::warning
Make sure the voltage polarity of the battery's connector matches with the voltage polarity of the connector on the WisBlock Base Board. The **+** wire from the battery must match the **+** sign next to the battery connector.
:::

> **Image:** Close the Lid

## Installation

1. Use a Ø5 mm drill head and drill two holes in the wall according to the dimensions shown in **Figure 8**:

> **Image:** Drill Holes

2. Install the screw anchor into the wall, then install the screw into the anchor, leaving about 3 mm between the screw head and wall.

> **Image:** Install Screws

3. Insert the screw head into the mounting hole at the back of the enclosure, then gently pull the enclosure downward to complete the installation.

> **Image:** Install Enclosure on Wall
