## Assembly

**Step 1:** Fix the core and sensor modules to the WisBlock Base with an appropriate number of M1.2*3 screws. For assembly methods and precautions, refer to the [WisBlock Base Board Installation Guide](https://learn.rakwireless.com/hc/en-us/articles/26743966497431-How-To-Install-RAK5005-O-Baseboard/).

:::tip NOTE
To optimize ambient monitoring, place the RAK1902 barometric pressure (for WisBlock Kit 1) on the bottom sensor slot, then place the light and temperature sensor on the top side.
:::

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/attaching-sensor.png"
  width="40%"
  caption="Assembly - Attaching Sensor"
/>

**Step 2:** Stick the foam to the position of the shield cover and IO slot on the WisBlock module. The foam is used as a base for the antenna later.

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/attaching-foam.png"
  width="40%"
  caption="Assembly - Attaching Foam for Antenna"
/>

**Step 3:** Fix the mounting plate with tapping screws on the enclosure.

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/wisblock-box1.png"
  width="40%"
  caption="Assembly - Attaching Screws"
/>


## Installation

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/box-screw-distance.png"
  width="25%"
  caption="2 Drill Holes"
/>

**Step 2**: Install the screw anchor into the wall.

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/wall-screw.png"
  width="40%"
  caption="Attaching to the Wall"
/>

**Step 3:** Install the device with two screws.

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/box-screw.png"
  width="40%"
  caption="Installing the Device in the Wall"
/>

**Step 4:** Install the battery with two double-sided adhesive foam.

:::tip NOTE
The dimension of the battery should not exceed 65 x 44 x 8&nbsp;mm.
:::

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/battery-install.png"
  width="40%"
  caption="Attaching Battery"
/>

**Step 5:** Install the WisBlock Base board with four M2.5*4 screws.

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/final-steps-1.png"
  width="40%"
  caption="Fix WisBlock Modules"
/>

**Step 6:** Stick the antenna to the foam and clip the antenna wire into the IPEX connector. Then, plug the terminal of the battery into the socket of the WisBlock Base.

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/final-steps-2.png"
  width="40%"
  caption="Attach the Antennas"
/>

:::warning
Make sure the voltage polarity of the battery's connector matches with the voltage polarity of the connector on the WisBlock Base Board. The **+** wire from the battery must match with the **+** sign next to the battery connector.
:::

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/battery-polarity.png"
  width="50%"
  caption="Battery Polarity"
/>

**Step 7:** Install the cover.

<RkImage
  src="https://images.docs.rakwireless.com/accessories/rakbox-b3/installation/box-cover.png"
  width="50%"
