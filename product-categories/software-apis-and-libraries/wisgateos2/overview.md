# WisGateOS 2 User Manual

Before using WisGateOS 2, ensure that your gateway has been physically installed and connected to a power source. Follow the setup instructions in the user manual for installation, powering on, and logging in to the Web UI.

## Overview of WisGateOS 2

This document provides a detailed overview of WisGateOS 2, an operating system built on OpenWrt and designed to run on RAK gateways. It covers general information, configuration guides, and detailed instructions for setting up the gateway.

Some sections may apply only to specific models due to differences in functionality. WisGateOS 2 enables efficient management of LoRaWAN networks and offers flexible features for IoT applications.

### Supported Gateway Models

The following table lists the gateway models supported by WisGateOS 2:

|          Product          |         Model          |
|:-------------------------:|:----------------------:|
|  **WisGate Edge Lite 2**  | RAK7268V2 / RAK7268CV2 |
|  **WisGate Edge Pro V2**  | RAK7289V2 / RAK7289CV2 |
| **WisGate Edge Prime V2** | RAK7240V2 / RAK7240CV2 |
|  **WisGate Edge Ultra**   |   RAK7285 / RAK7285C   |
|   **WisGate Soho Pro**    |        RAK7267         |
|   **WisGate Soho Lite**   |        RAK7266         |

:::tip NOTE
WisGateOS 2 is compatible only with V2 hardware models. V1 models are not supported due to hardware differences.
 :::

### Supported Features

This section outlines the key capabilities supported by **WisGateOS 2**.

+ **Flexible LoRaWAN® Operation Modes** – Supports Built-in Network Server (LNS), Packet Forwarder, and secure Basics™ Station.
+ **Cloud & Platform Integrations** – Enables seamless MQTT and HTTP/HTTPS forwarding for third-party data visualization and control.
+ **Reliable Network Connectivity** – Multi-WAN support with Ethernet, LTE, and Wi-Fi, including automatic backup and failover.
+ **Extensible Architecture** – Expand functionality with modular **Extensions** such as OpenVPN Client, WireGuard VPN, Failover, or Field Test Data Processor.
+ **Advanced Gateway Management** – Remote configuration, mass deployment, and live monitoring through **WisDM**.

For detailed configuration and feature guides, and to explore many more features we support, visit the [RAKwireless Learn – How To](https://learn.rakwireless.com/hc/en-us/sections/26418821514519-How-To) section.
